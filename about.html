What is Terp Transit Finder

Terp Transit Finder was made to help visualize the location nad information regarding UMD buildings. The Map simply displays a random list of 35 buildings at a time and shows markers with their corresponding information.

Where is the data from?

The visualizer uses the UMD API that contains all sorts of information regarding the university from bus stops to majors to building information.

How to Use

To Use the following Visualizer, simply press load UMD building data, and then generate list. At this point you will get back a list of 35 items which you can see the information for in the Map section.
